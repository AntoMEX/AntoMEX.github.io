Lenguaje HTML

El Lenguaje de Marcación de Hipertexto o Hyper Text Markup Language permite presentar la información y objetos como tablas, botones, enlaces, listas, formularios u otros elementos dentro de la Web.

Variantes del lenguaje HTML

  1. DHTML
  2. XHTML
  3. PHP
